# Kompi

**Kompi** is a Khmer programming learning platform that provides programming lessons in the Khmer language. The goal of Kompi is to help Cambodian students, beginners, and self-learners understand programming more easily through Khmer explanations, examples, and practice projects.

## About Kompi

Kompi provides lessons and learning resources for different programming languages and software development topics. It is designed to make coding education more accessible for Khmer-speaking learners.

## Features

* Programming lessons in Khmer
* Beginner-friendly explanations
* Code examples and exercises
* Web development lessons
* Software development documentation
* Project roadmap and learning guide
* Open contribution for the community

## Lessons

Kompi will include lessons such as:

* HTML
* CSS
* JavaScript
* PHP
* Laravel
* Python
* Java
* C / C++
* SQL and Database
* Web Development
* Mobile App Development
* Git and GitHub

## Project Goal

The main goal of Kompi is to become a complete Khmer programming resource where learners can study, practice, and improve their coding skills step by step.
