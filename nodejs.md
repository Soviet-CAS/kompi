### 1. Introduction to Node.js

### бҹЎ. бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһўбҹҶбһ–бһё Node.js
Node.js is a JavaScript runtime built on Chrome's V8 engine for server-side development.

Node.js бһӮбһәбһҮбһ¶бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ JavaScript runtime бһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһӣбһҫ Chrome's V8 engine бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖ serverбҹ”

#### Example

```javascript
// First Node.js program
console.log('Hello, Node.js!');
console.log('Node.js version:', process.version);
```

#### Explanation
бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһҹбһ¶бһҡбһҹбһ¶бһҳбһүбҹ’бһү 'Hello, Node.js!' бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹҶбһҺбҹӮ Node.js бһҠбҹӮбһӣбһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбҹ” `process.version` бһ•бҹ’бһҸбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбҹҶбһ–бһёбһҖбҹҶбһҺбҹӮ Node.jsбҹ”

#### Output

```text
Hello, Node.js!
Node.js version: v18.17.0
```

#### Summary / бһҹбһҡбһ»бһ”
бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһҺбҹӮбһ“бһ¶бҹҶ Node.js бһҠбҹӮбһӣбһҮбһ¶бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ JavaScript бһ“бҹ…бһ•бҹ’бһ“бҹӮбһҖ serverбҹ” бһңбһ¶бһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһӮбҹ„бһӣбһӮбҹҶбһ“бһ·бһҸбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Node.js бһҠбҹҶбһ”бһјбһ„бҹ”

### 2. Installing Node.js & npm

### бҹў. бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ Node.js бһ“бһ·бһ„ npm
Guide to installing Node.js and npm on various operating systems.

бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ Node.js бһ“бһ·бһ„ npm бһ“бҹ…бһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

#### Example


#### Explanation
бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹҶбһЎбһҫбһ„ Node.js бһ“бһ·бһ„ npm бһ“бҹ…бһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Ubuntu/Debian бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `apt`бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҖбҹҶбһҺбҹӮбһҠбҹӮбһӣбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `node --version` бһ“бһ·бһ„ `npm --version`бҹ”

#### Output

```text
v18.17.0
9.6.7
```

#### Summary / бһҹбһҡбһ»бһ”
бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһҺбҹӮбһ“бһ¶бҹҶбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹҶбһЎбһҫбһ„ Node.js бһ“бһ·бһ„ npm бһҠбҹӮбһӣбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһүбҹ’бһ…бһ”бҹӢ бһ“бҹ…бһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ”

### 3. Running Your First Node.js Program

### бҹЈ. бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Node.js бһҠбҹҶбһ”бһјбһ„
Create and run a simple Node.js program.

бһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Node.js бһҹбһ¶бһҳбһүбҹ’бһүбҹ”

#### Example

```javascript
// app.js
console.log('Hello, Node.js World!');
console.log('Current time:', new Date().toLocaleString());
```

#### Explanation
бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡ `app.js` бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһҹбһ¶бһҡ 'Hello, Node.js World!' бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `new Date().toLocaleString()`бҹ” бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `node app.js`бҹ”

#### Output

```text
Hello, Node.js World!
Current time: 8/18/2025, 4:19:00 PM
```

#### Summary / бһҹбһҡбһ»бһ”
бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Node.js бһҠбҹҶбһ”бһјбһ„ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҹбһ¶бһҡбһҹбһ¶бһҳбһүбҹ’бһү бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶бҹ”

### 4. Understanding Modules & require()

### бҹӨ. бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһҳбҹүбһјбһҢбһ»бһӣ бһ“бһ·бһ„ require()
Learn about Node.js modules and the require() function.

бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһёбһҳбҹүбһјбһҢбһ»бһӣ Node.js бһ“бһ·бһ„бһҳбһ»бһҒбһ„бһ¶бһҡ require()бҹ”

#### Example

```javascript
// math.js
exports.add = (a, b) => a + b;

// app.js
const math = require('./math');
console.log(math.add(2, 3));
```

#### Explanation
бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҢбһ»бһӣ `math.js` бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбһ»бһҒбһ„бһ¶бһҡбһ”бһјбһҖбһӣбҹҒбһҒбҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `app.js` бһҷбһҫбһ„бһ“бһ¶бҹҶбһ…бһјбһӣбһҳбҹүбһјбһҢбһ»бһӣбһ“бҹҒбҹҮбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `require()` бһ“бһ·бһ„бһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ `add` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһјбһҖ 2 бһ“бһ·бһ„ 3бҹ”

#### Output

```text
5
```

#### Summary / бһҹбһҡбһ»бһ”
бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһ–бһ“бҹ’бһҷбһӣбҹӢбһ–бһёбһҳбҹүбһјбһҢбһ»бһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ `require()` бһҠбһҫбһҳбҹ’бһ”бһёбһ“бһ¶бҹҶбһ…бһјбһӣбһҳбһ»бһҒбһ„бһ¶бһҡбһ–бһёбһҜбһҖбһҹбһ¶бһҡбһҳбһҪбһҷбһ‘бҹ…бһҜбһҖбһҹбһ¶бһҡбһҳбһҪбһҷбһ‘бҹҖбһҸбҹ”

### 5. Node.js REPL (Read-Eval-Print-Loop)

### бҹҘ. Node.js REPL (бһўбһ¶бһ“-бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғ-бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–-бһҡбһ„бҹ’бһңбһ·бһӣбһҮбһ»бҹҶ)
Use the Node.js REPL for interactive JavaScript coding.

бһ”бҹ’бһҡбһҫ Node.js REPL бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ JavaScript бһҠбҹ„бһҷбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбҹ”

#### Example

```bash
# Start REPL
node
> console.log('Hello from REPL!');
> .exit
```

#### Explanation
бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ REPL бһҠбҹ„бһҷбһңбһ¶бһҷ `node` бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ terminalбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһҷбһҫбһ„бһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ `console.log` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҹбһ¶бһҡ бһ“бһ·бһ„бһ”бҹ’бһҡбһҫ `.exit` бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһҖбһ…бҹҒбһүбҹ”

#### Output

```text
Hello from REPL!
```

#### Summary / бһҹбһҡбһ»бһ”
бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһҺбҹӮбһ“бһ¶бҹҶбһ–бһё REPL бһҠбҹӮбһӣбһҮбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ“бҹ’бһҸбһҡбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһјбһҠ JavaScript бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Node.jsбҹ”

### 6. File System (fs) Module

### бҹҰ. бһҳбҹүбһјбһҢбһ»бһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҜбһҖбһҹбһ¶бһҡ (fs)
Work with the file system using the fs module.

бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҜбһҖбһҹбһ¶бһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбһ»бһӣ fsбҹ”

#### Example

```javascript
const fs = require('fs');
fs.writeFileSync('example.txt', 'Hello, Node.js!');
const data = fs.readFileSync('example.txt', 'utf8');
console.log(data);
```

#### Explanation
бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбһ»бһӣ `fs` бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҡбһҹбҹҒбһҡбһҹбһ¶бһҡ 'Hello, Node.js!' бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `example.txt` бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `writeFileSync`бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖ бһўбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ `readFileSync` бһ“бһ·бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҒбҹ’бһӣбһ№бһҳбһҹбһ¶бһҡбҹ”

#### Output

```text
Hello, Node.js!
```

#### Summary / бһҹбһҡбһ»бһ”
бһҮбҹҶбһ–бһјбһҖбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбһ»бһӣ `fs` бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ“ бһ“бһ·бһ„бһҹбһҡбһҹбҹҒбһҡбһҜбһҖбһҹбһ¶бһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Node.jsбҹ”

### 7. Path Module

### бҹ§. бһҳбҹүбһјбһҢбһ»бһӣ Path

### 13. Middleware in Express

### бҹЎбҹЈ. Middleware бһҖбҹ’бһ“бһ»бһ„ Express
Use middleware to handle requests in Express.

бһ”бҹ’бһҡбһҫ middleware бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹҶбһҺбһҫбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Expressбҹ”

#### Example

```javascript
const express = require('express');
const app = express();
const logger = (req, res, next) => {
  console.log(`${req.method} ${req.url}`);
  next();
};
app.use(logger);
app.get('/', (req, res) => res.send('Hello!'));
app.listen(3000);
```

#### Explanation
бһҖбһјбһҠбһ“бҹҒбҹҮбһ”бһ„бҹ’бһҖбһҫбһҸ middleware `logger` бһҠбҹӮбһӣбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ HTTP бһ“бһ·бһ„ URL бһ“бҹғбһҹбҹҶбһҺбһҫбҹ” `app.use` бһўбһ“бһ»бһңбһҸбҹ’бһҸ middleware бһ‘бҹ…бһӮбҹ’бһҡбһ”бҹӢбһҹбҹҶбһҺбһҫ бһ“бһ·бһ„ `next()` бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһ”бһ“бҹ’бһҸбһ‘бҹ… route бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбҹ”

Example
```javascript

  }

```
