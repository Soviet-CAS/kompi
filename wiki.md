# Welcome to Kompi

Welcome to **Kompi**, a Khmer learning platform created to provide programming lessons for everyone who wants to learn software development in the Khmer language.

Kompi is designed to help beginners and developers learn programming step by step through clear explanations, practical examples, and real project guidance.

## About Kompi

Kompi provides lessons for many programming languages and software development topics in Khmer, including:

* HTML, CSS, and JavaScript
* PHP and Laravel
* Python
* Java
* C and C++
* Database and SQL
* Web development
* Mobile app development
* Software project practice

Our goal is to make programming easier to understand for Cambodian learners by using Khmer language, simple examples, and practical exercises.

## Project Roadmap

Kompi aims to grow step by step with the following goals:

### Stage 1: Basic Learning Content

Create beginner-friendly programming lessons in Khmer for core technologies such as HTML, CSS, JavaScript, PHP, and Python.

### Stage 2: Practice Projects

Add practical exercises and small projects so learners can apply what they study.

### Stage 3: Advanced Lessons

Provide more advanced topics such as frameworks, databases, APIs, authentication, deployment, and software architecture.

### Stage 4: Community Contribution

Allow developers, teachers, and students to contribute lessons, examples, documentation, and improvements.

### Stage 5: Complete Khmer Programming Resource

Build Kompi into a complete Khmer programming knowledge base for students, self-learners, and developers.
