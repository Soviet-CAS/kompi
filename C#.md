# 1. Introduction to C#

C# (бһўбһ¶бһ“бһҗбһ¶ "C Sharp") бһӮбһәбһҮбһ¶бһ—бһ¶бһҹбһ¶бһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бҹҶбһ“бһҫбһ”бһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһңбһҸбҹ’бһҗбһ»бһ“бһ·бһҷбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷ Microsoftбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Windows, бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”, бһ бҹ’бһӮбҹҒбһҳ (бһҸбһ¶бһҳбһҡбһҷбҹҲ Unity), бһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бҹ” C# бһӮбһәбһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ .NET бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһҖбҹ’бһҡбһ”бһҒбҹҗбһҺбҹ’бһҢбһҠбҹҸбһҡбһ№бһ„бһҳбһ¶бҹҶбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбҹ’бһҗбҹҒбһҡбһ—бһ¶бһ–бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бҹ”

## 1.1 What is C#?

C# бһӮбһәбһҮбһ¶бһ—бһ¶бһҹбһ¶бһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷ Microsoft бһҖбҹ’бһ“бһ»бһ„бһҶбҹ’бһ“бһ¶бҹҶ 2000бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ“бҹ…бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶ .NETбҹ” C# бһӮбһәбһҮбһ¶бһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҡбҹҖбһ“бһ“бһ·бһ„бһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҠбһјбһ…бһҮбһ¶ OOP (Object-Oriented Programming), type safety, бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһ бһ»бһҹбҹ”

бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҡбһ”бһҹбҹӢ C#бҹ–

- бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹӮбһ”бһңбһҸбҹ’бһҗбһ» (OOP)бҹ”

- бһҳбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ”бҹ’бһҡбһҳбһјбһӣбһҹбҹҶбһҡбһ¶бһҳбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· (Garbage Collection)бҹ”

- бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһӣбһҫбһңбҹҒбһ‘бһ·бһҖбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ— (Windows, macOS, Linux)бҹ”

- бһҳбһ¶бһ“бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҠбҹӮбһӣбһ’бҹҶбһ‘бһјбһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ OOP бһҖбҹ’бһ“бһ»бһ„ C#бҹ–

```csharp
using System;

class Animal {
    public string Name { get; set; }

    public void Speak() {
        Console.WriteLine(Name + " makes a sound.");
    }
}

class Dog : Animal {
    public Dog(string name) {
        Name = name;
    }

    public void Bark() {
        Console.WriteLine(Name + " barks.");
    }
}

class Program {
    static void Main() {
        Dog dog = new Dog("Buddy");
        dog.Speak(); // Output: Buddy makes a sound.
        dog.Bark();  // Output: Buddy barks.
    }
}
```

**бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–**

```text
Buddy makes a sound.
Buddy barks.
```

## 1.4 Understanding the .NET Framework

.NET бһӮбһәбһҮбһ¶бһҖбҹ’бһҡбһ”бһҒбҹҗбһҺбҹ’бһҢбһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ“бһ·бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ” бһңбһ¶бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ CLR (Common Language Runtime) бһ“бһ·бһ„ FCL (Framework Class Library)бҹ”

бһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ»бһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—бһ“бҹғ .NETбҹ–

- CLR: бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһјбһҠбһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбҹ”

- FCL: бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбҹ”

- бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бһҳбҹ’бһҡбһ»бҹҮ (C#, F#, VB.NET)бҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ .NETбҹ–

```csharp
using System;

class Program {
    static void Main() {
        DateTime now = DateTime.Now;
        Console.WriteLine("Current date and time: " + now);
    }
}
```

**бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–**

```text
Current date and time: 10/25/2023 12:34:56 PM
```

# 2. C# Basics

бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһўбҹҶбһ–бһёбһӮбҹ„бһӣбһӮбҹҶбһ“бһ·бһҸбһҮбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё C# бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„ syntax, бһўбһҗбҹҒбһҡ, бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ, бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡ, бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” бһӮбҹ„бһӣбһӮбҹҶбһ“бһ·бһҸбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһӮбҹ’бһҡбһ№бҹҮбһҡбһ№бһ„бһҳбһ¶бҹҶбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ C#бҹ”

## 2.1 Syntax and Structure

C# syntax бһӮбһәбһҹбҹ’бһҡбһҠбҹҖбһ„бһ“бһ№бһ„бһ—бһ¶бһҹбһ¶ C-style бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҠбһјбһ…бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһңбһӮбҹ’бһӮбһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһўбһ“бһ»бһӮбһҳбһ“бҹҚ Main()бҹ”

бһӮбһ“бҹ’бһӣбһ№бҹҮ Syntax бһҡбһ”бһҹбҹӢ C#бҹ–

- бһҡбһ¶бһӣбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё C# бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҳбһ¶бһ“бһўбһ“бһ»бһӮбһҳбһ“бҹҚ Main() бһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбҹ”

- бһ”бҹ’бһҡбһҫ using бһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢ namespaceбҹ”

- бһ”бҹ’бһҡбһҫ ; бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҗбҹ’бһӣбҹӮбһ„бһҖбһ¶бһҡбһҺбҹҚбҹ”

- бһ”бҹ’бһҡбһҫ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһ”бҹ’бһӣбһ»бһҖбһҖбһјбһҠбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё C# бһҹбһ¶бһҳбһүбҹ’бһүбҹ–

```csharp
using System;

class Program {
    static void Main() {
        Console.WriteLine("Hello, C#!");
    }
}
```

**бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–**

```text
Hello, C#!
```

## 2.2 Variables and Data Types

бһўбһҗбҹҒбһҡбһӮбһәбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бһ·бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” C# бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбһјбһ…бһҮбһ¶ int, double, string, bool, бһ“бһ·бһ„бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸбҹ”

бһҖбһ¶бһҡбһ”бҹ’бһҡбһҖбһ¶бһҹбһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһўбһҗбҹҒбһҡ

бһўбһҗбҹҒбһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҖбһ¶бһҹбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҲбҹ’бһҳбҹ„бҹҮбһўбһҗбҹҒбһҡбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһўбһҗбҹҒбһҡбһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһҖбһ¶бһҹ бһ¬бһҖбҹ’бһҡбҹ„бһҷбһҳбһҖбҹ”

```csharp
// бһ”бҹ’бһҡбһҖбһ¶бһҹбһўбһҗбҹҒбһҡ
int age;
double salary;
string name;
bool isActive;

// бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһўбһҗбҹҒбһҡ
age = 25;
salary = 1000.50;
name = "John";
isActive = true;

// бһ”бҹ’бһҡбһҖбһ¶бһҹбһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһӣбҹғбһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҸбҹӮбһҳбһҪбһҷ
int score = 100;
double pi = 3.14159;
string greeting = "Hello, World!";
bool isCompleted = false;
```

бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ C#

C# бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бҹ” бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһӮбҹҒбһ”бҹ’бһҡбһҫбһүбһ№бһҖбһүбһ¶бһ”бҹӢбҹ–

1. бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢ (Integer Types)

бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢбһҡбһҪбһҳбһҳбһ¶бһ“ int, long, short, бһ“бһ·бһ„ byteбҹ”

int age = 25; // бһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢ (32-bit)

long population = 7800000000; // бһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢбһ’бҹҶ (64-bit)

short temperature = -10; // бһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢбһҸбһјбһ… (16-bit)

byte flags = 0b1010; // бһ…бҹҶбһ“бһҪбһ“бһӮбһҸбҹӢбһҸбһјбһ… (8-bit)

2. бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ…бҹҶбһ“бһҪбһ“бһ‘бһҹбһ—бһ¶бһӮ (Floating-Point Types)

бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ…бҹҶбһ“бһҪбһ“бһ‘бһҹбһ—бһ¶бһӮбһҡбһҪбһҳбһҳбһ¶бһ“ float, double, бһ“бһ·бһ„ decimalбҹ”

float piFloat = 3.14f; // бһ…бҹҶбһ“бһҪбһ“бһ‘бһҹбһ—бһ¶бһӮ (32-bit)

double piDouble = 3.14159; // бһ…бҹҶбһ“бһҪбһ“бһ‘бһҹбһ—бһ¶бһӮ (64-bit)

decimal price = 19.99m; // бһ…бҹҶбһ“бһҪбһ“бһ‘бһҹбһ—бһ¶бһӮбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбһҳбҹ’бһӣбҹғбһ бһ·бһҡбһүбҹ’бһүбһңбһҸбҹ’бһҗбһ» (128-bit)

3. бһ”бҹ’бһҡбһ—бҹҒбһ‘бһўбһҖбҹ’бһҹбһҡ бһ“бһ·бһ„бһғбҹ’бһӣбһ¶ (Character and String Types)

бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһўбһҖбҹ’бһҹбһҡбһҡбһҪбһҳбһҳбһ¶бһ“ char бһ“бһ·бһ„ stringбҹ”

char grade = 'A'; // бһўбһҖбҹ’бһҹбһҡбһҸбҹӮбһҳбһҪбһҷ

string name = "John Doe"; // бһғбҹ’бһӣбһ¶бһўбһҖбҹ’бһҹбһҡ

4. бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһҸ (Boolean Type)

бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ bool бһ•бҹ’бһ‘бһ»бһҖбһҸбһҳбҹ’бһӣбҹғбһ–бһ·бһҸ (true) бһ¬бһҳбһ·бһ“бһ–бһ·бһҸ (false)бҹ”

```csharp
bool isActive = true;
bool isCompleted = false;
```

5. бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ— (Other Data Types)

C# бһҖбҹҸбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бҹҖбһҸ бһҠбһјбһ…бһҮбһ¶ object, dynamic, бһ“бһ·бһ„ varбҹ”

object obj = 10; // бһўбһ¶бһ…бһ•бҹ’бһ‘бһ»бһҖбһҸбһҳбҹ’бһӣбҹғбһҺбһ¶бһҳбһҪбһҷ

dynamic dynamicValue = "Hello"; // бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ“бҹ…бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ

var message = "Hello, World!"; // бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·

бһўбһҗбҹҒбһҡбһҗбҹҒбһҡ (Constants)

бһўбһҗбҹҒбһҡбһҗбҹҒбһҡбһӮбһәбһҮбһ¶бһўбһҗбҹҒбһҡбһҠбҹӮбһӣбһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһҳбҹ’бһӣбҹғбһ”бһ¶бһ“бҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҖбһ¶бһҹбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ constбҹ”

```csharp
const double PI = 3.14159;
const string GREETING = "Hello, World!";
```

## 2.3 Constants and Readonly Fields

Constants бһ“бһ·бһ„ readonly fields бһӮбһәбһҮбһ¶бһўбһҗбҹҒбһҡбһҠбҹӮбһӣбһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһҳбҹ’бһӣбҹғбһ”бһ¶бһ“бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбҹ” Constants бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ const бһҒбһҺбҹҲбһ–бҹҒбһӣбһҠбҹӮбһӣ readonly fields бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ readonlyбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбҹ–

```csharp
const double PI = 3.14159; // Constant value
readonly int maxAttempts = 5; // Readonly field (can be set in constructor)
```

**бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–**

```text
PI = 3.14159
maxAttempts = 5
```

бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„ const бһ“бһ·бһ„ readonlyбҹ–

- const: бһҸбһҳбҹ’бһӣбҹғбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҖбҹҶбһҺбһҸбҹӢбһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһҖбһ¶бһҹ бһ“бһ·бһ„бһҳбһ·бһ“бһўбһ¶бһ…бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ”бһ¶бһ“бҹ”

- readonly: бһҸбһҳбҹ’бһӣбҹғбһўбһ¶бһ…бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһҖбһ¶бһҹ бһ¬бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ constructor бһ“бҹғ classбҹ”

## 2.4 Operators (Arithmetic, Logical, Comparison)

Operators бһӮбһәбһҮбһ¶бһ“бһ·бһҳбһ·бһҸбҹ’бһҸбһҹбһүбҹ’бһүбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһӣбһҫбһҸбһҳбҹ’бһӣбҹғбһ“бһ·бһ„бһўбһҗбҹҒбһҡбҹ” C# бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһҡбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһјбһ…бһҮбһ¶ arithmetic, logical, бһ“бһ·бһ„ comparisonбҹ”

### Arithmetic Operators

Arithmetic operators бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһӮбһҺбһ·бһҸбһңбһ·бһ‘бҹ’бһҷбһ¶бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҠбһјбһ…бһҮбһ¶ бһ”бһјбһҖ, бһҠбһҖ, бһӮбһ»бһҺ, бһ“бһ·бһ„бһ…бҹӮбһҖбҹ”

```csharp
int a = 10;
int b = 5;

int sum = a + b;       // Addition, sum = 15
int difference = a - b; // Subtraction, difference = 5
int product = a * b;    // Multiplication, product = 50
int quotient = a / b;   // Division, quotient = 2
int remainder = a % b;  // Modulus, remainder = 0
```

**бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–**

```text
sum = 15
difference = 5
product = 50
quotient = 2
remainder = 0
Comparison Operators
```

Comparison operators бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбҹҖбһ”бһ’бҹҖбһ”бһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҡбҹ” бһӣбһ‘бҹ’бһ’бһ•бһӣбһӮбһәбһҮбһ¶бһҸбһҳбҹ’бһӣбҹғ boolean (`true` бһ¬ `false`)бҹ”

```csharp
int x = 10;
int y = 5;

bool isEqual = (x == y);      // Equal to, isEqual = false
bool isNotEqual = (x != y);   // Not equal to, isNotEqual = true
bool isGreater = (x > y);     // Greater than, isGreater = true
bool isLess = (x < y);        // Less than, isLess = false
bool isGreaterOrEqual = (x >= y); // Greater than or equal to, isGreaterOrEqual = true
bool isLessOrEqual = (x <= y);    // Less than or equal to, isLessOrEqual = false
```

**бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–**

```text
isEqual = False
isNotEqual = True
isGreater = True
isLess = False
isGreaterOrEqual = True
isLessOrEqual = False
Logical Operators
```

## 4.4 Recursion

Recursion бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһ бҹ… method бһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„бҹ” бһңбһ¶бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҠбһјбһ…бһҮбһ¶ factorial бһ¬ Fibonacciбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ recursionбҹ–

```csharp
int Factorial(int n) {
    if (n == 0) {
        return 1;
    }
    return n * Factorial(n - 1);
}

int result = Factorial(5);
Console.WriteLine(result);
```

```csharp
int Fibonacci(int n) {
    if (n <= 1) {
        return n;
    }
    return Fibonacci(n - 1) + Fibonacci(n - 2);
}

int result = Fibonacci(6);
Console.WriteLine(result);
```

# 5. Object-Oriented Programming (OOP) in C#

## 5.1 Classes and Objects

Class бһӮбһәбһҮбһ¶бһӮбҹ’бһҡбҹ„бһ„бһҖбһ¶бһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸ objects бһҒбһҺбҹҲбһ–бҹҒбһӣ object бһӮбһәбһҮбһ¶ instance бһ“бҹғ classбҹ” Class бһҳбһ¶бһ“ fields (бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ) бһ“бһ·бһ„ methods (бһўбһ“бһ»бһӮбһҳбһ“бҹҚ)бҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ class бһ“бһ·бһ„ objectбҹ–

```csharp
class Car {
    public string Model; // Field
    public void Drive() { // Method
        Console.WriteLine("Driving " + Model);
    }
}

Car myCar = new Car(); // Create an object
myCar.Model = "Toyota";
myCar.Drive();
```

## 5.2 Constructors and Destructors

Constructor бһӮбһәбһҮбһ¶бһўбһ“бһ»бһӮбһҳбһ“бҹҚбһ–бһ·бһҹбҹҒбһҹбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһ“бҹ…бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ objectбҹ” Destructor бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһ“бҹ…бһ–бҹҒбһӣ object бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ•бҹ’бһӣбһ¶бһүбҹ”

```csharp
class Car {
    public string Model;

    // Constructor
    public Car(string model) {
        Model = model;
        Console.WriteLine("Car created: " + Model);
    }

    // Destructor
    ~Car() {
        Console.WriteLine("Car destroyed: " + Model);
    }
}

Car myCar = new Car("Toyota");
```

```text
Car created: Toyota
```

Note: Destructor бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһ“бҹ…бһ–бҹҒбһӣ object бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹҶбһ•бҹ’бһӣбһ¶бһү (e.g., бһ“бҹ…бһ–бҹҒбһӣбһ…бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё)бҹ”

## 6.1 Try-Catch-Finally Blocks

try-catch-finally бһӮбһәбһҮбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһ¶бһ”бҹӢбһҷбһҖ exceptionsбҹ” бһҖбһјбһҠбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ try block бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸ бһ бһҫбһҷбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“ exception бһңбһ¶бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ…бһ¶бһ”бҹӢбһҷбһҖбһҠбҹ„бһҷ catch blockбҹ”finally block бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ try-catch-finallyбҹ–

```csharp
try {
    int result = 10 / 0; // This will throw an exception
} catch (DivideByZeroException ex) {
    Console.WriteLine("Error: " + ex.Message);
} finally {
    Console.WriteLine("This will always execute.");
}
```

**бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–**

```text
Error: Attempted to divide by zero.
This will always execute.
```

## 6.5 Nested Try-Catch Blocks

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ try-catch blocks бһҠбҹӮбһӣбһҮбһ¶бһ“бҹӢбһӮбҹ’бһ“бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ exceptions бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ nested try-catch blocksбҹ–

```csharp
try {
    try {
        int result = 10 / 0; // This will throw a DivideByZeroException
    } catch (DivideByZeroException ex) {
        Console.WriteLine("Inner error: " + ex.Message);
        throw; // Re-throw the exception
    }
} catch (Exception ex) {
    Console.WriteLine("Outer error: " + ex.Message);
}
```

**бһӣбһ‘бҹ’бһ’бһ•бһӣбҹ–**

```text

Outer error: Attempted to divide by zero.
```

Arrays бһӮбһәбһҮбһ¶бһ”бһҺбҹ’бһҸбһ»бҹҶбһ“бҹғбһ’бһ¶бһҸбһ»бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбһјбһ…бһӮбҹ’бһ“бһ¶бҹ” C# бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ arrays бһҳбһҪбһҷбһңбһ·бһҳбһ¶бһҸбҹ’бһҡ, бһ–бһ бһ»бһңбһ·бһҳбһ¶бһҸбҹ’бһҡ, бһ“бһ·бһ„ jagged arraysбҹ”

```csharp
// One-dimensional array

    Console.Write(num + " ");
}

int[,] matrix = { { 1, 2 }, { 3, 4 } };

for (int i = 0; i < 2; i++) {
    for (int j = 0; j < 2; j++) {
        Console.Write(matrix[i, j] + " ");

int[][] jaggedArray = {
    new int[] { 1, 2 },
    new int[] { 3, 4, 5 }
};

foreach (int[] arr in jaggedArray) {
    foreach (int num in arr) {
        Console.Write(num + " ");
    }
    Console.WriteLine();
}
```

## 8.1 Reading and Writing Files (File, StreamReader, StreamWriter)

C# бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһңбһ·бһ’бһёбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ¶бһ“бһ“бһ·бһ„бһҹбһҡбһҹбҹҒбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҜбһҖбһҹбһ¶бһҡбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ File class бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҹбһ¶бһҳбһүбҹ’бһү бһ¬ StreamReader бһ“бһ·бһ„ StreamWriter бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбһҳбҹ’бһўбһ·бһҸбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғбһҖбһ¶бһҡбһҹбһҡбһҹбҹҒбһҡбһ“бһ·бһ„бһўбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбҹ–

```csharp
using System;
using System.IO;

class Program
{
    static void Main()
    {
        string filePath = "example.txt";

        // Writing to a file using File.WriteAllText
        File.WriteAllText(filePath, "Hello, World!");
        Console.WriteLine("File written using File.WriteAllText.");

        // Reading from a file using File.ReadAllText
        string content = File.ReadAllText(filePath);
        Console.WriteLine("Content read using File.ReadAllText:");
        Console.WriteLine(content); // Output: Hello, World!

        // Appending to a file using File.AppendAllText
        File.AppendAllText(filePath, "\nThis is an appended line.");
        Console.WriteLine("Line appended using File.AppendAllText.");

        // Reading all lines from a file using File.ReadAllLines
        string[] lines = File.ReadAllLines(filePath);
        Console.WriteLine("Content read using File.ReadAllLines:");
        foreach (string line in lines)
        {
            Console.WriteLine(line);
        }
        // Output:
        // Hello, World!
        // This is an appended line.

        // Using StreamWriter to write to a file
        using (StreamWriter writer = new StreamWriter(filePath))
        {
            writer.WriteLine("This is a new line written using StreamWriter.");
            writer.WriteLine("Another line added.");
        }
        Console.WriteLine("File written using StreamWriter.");

        // Using StreamReader to read from a file line by line
        using (StreamReader reader = new StreamReader(filePath))
        {
            Console.WriteLine("Content read using StreamReader:");
            string line;
            while ((line = reader.ReadLine()) != null)
            {
                Console.WriteLine(line);
            }
        }
        // Output:
        // This is a new line written using StreamWriter.
        // Another line added.

        // Handling exceptions when working with files
        try
        {
            string nonExistentFilePath = "nonexistent.txt";
            string text = File.ReadAllText(nonExistentFilePath);
        }
        catch (FileNotFoundException ex)
        {
            Console.WriteLine("Error: File not found.");
            Console.WriteLine(ex.Message);
        }
        catch (Exception ex)
        {
            Console.WriteLine("An error occurred:");
            Console.WriteLine(ex.Message);
        }
    }
}
```

**Output:**

```text
File written using File.WriteAllText.
Content read using File.ReadAllText:
Hello, World!
Line appended using File.AppendAllText.
Content read using File.ReadAllLines:
Hello, World!
This is an appended line.
File written using StreamWriter.
Content read using StreamReader:
This is a new line written using StreamWriter.
Another line added.
Error: File not found.
Could not find file 'nonexistent.txt'.
```

## 8.2 Working with JSON (System.Text.Json)

JSON (JavaScript Object Notation) бһӮбһәбһҮбһ¶бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ¶бһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһӮбҹҒбһ”бҹ’бһҡбһҫбһҷбҹүбһ¶бһ„бһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ” C# бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһң System.Text.Json бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„ objects бһ‘бҹ…бһҮбһ¶ JSON бһ“бһ·бһ„бһ…бҹ’бһҡбһ¶бһҹбһҳбһҖбһңбһ·бһүбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ JSONбҹ–

```csharp
using System.Text.Json;

// Define a Person class
public class Person {
    public string Name { get; set; }
    public int Age { get; set; }
}

// Serialize object to JSON
var person = new Person { Name = "John", Age = 30 };
string json = JsonSerializer.Serialize(person);
Console.WriteLine(json); // Output: {"Name":"John","Age":30}

// Deserialize JSON to object
var deserializedPerson = JsonSerializer.Deserialize<Person>(json);
Console.WriteLine(deserializedPerson.Name); // Output: John
Console.WriteLine(deserializedPerson.Age);  // Output: 30
```

**Output:**

```text
{"Name":"John","Age":30}
John
30
```

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ CSVбҹ–

## 10.1 Delegates (Func, Action, Predicate)

Delegates бһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ—бҹҒбһ‘бһҠбҹӮбһӣбһҸбҹҶбһҺбһ¶бһ„бһұбҹ’бһҷбһўбһ“бһ»бһӮбһҳбһ“бҹҚбҹ” C# бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһң delegates бһҠбһјбһ…бһҮбһ¶ Func, Action, бһ“бһ·бһ„ Predicate бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһјбһ‘бҹ…бҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ delegatesбҹ–

```csharp
// Func delegate (returns a value)

// Action delegate (does not return a value)
Action<string> greet = name => Console.WriteLine("Hello, " + name);

// Predicate delegate (returns a boolean)
```

```text
Hello, John
True
```

## 10.3 Events and Event Handling

Events бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһұбҹ’бһҷ class бһўбһ¶бһ…бһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„бһ‘бҹ…бһҖбһ¶бһ“бҹӢ class бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһңбһёбһҳбһҪбһҷбһҖбһҫбһҸбһЎбһҫбһ„бҹ” Event handling бһӮбһәбһҮбһ¶бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ‘бҹ…бһ“бһ№бһ„ eventsбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ events бһ“бһ·бһ„ event handlingбҹ–

```csharp
// Define a delegate for the event
public delegate void Notify();

// Define a class with an event
public class Process {
    public event Notify ProcessCompleted;

    public void StartProcess() {
        Console.WriteLine("Process started...");
        OnProcessCompleted();
    }

    protected virtual void OnProcessCompleted() {
        ProcessCompleted?.Invoke();
    }
}

// Subscribe to the event
Process process = new Process();
process.ProcessCompleted += () => Console.WriteLine("Process completed!");
process.StartProcess(); // Output: Process started... Process completed!
```

**Output:**

```text
Process started...
Process completed!
```

## 11.1 Threads (Thread Class)

Threads бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһјбһҠбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҠбҹҶбһҺбһ¶бһӣбһӮбҹ’бһ“бһ¶бҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ Thread class бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ threadsбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ background workersбҹ–

```csharp
    e.Result = "Task completed!";

worker.RunWorkerCompleted += (sender, e) => {
    Console.WriteLine(e.Result); // Output: Task completed!
};

worker.RunWorkerAsync();
```

**Output:**


## 12.4 Entity Framework Core Basics

Entity Framework Core (EF Core) бһӮбһәбһҮбһ¶ ORM (Object-Relational Mapper) бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ databases бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ objects бһ“бһ·бһ„ LINQбҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһ¶бһҡбһ„бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ databases бһҖбһ¶бһ“бҹӢбһҸбҹӮбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ EF Coreбҹ–

```csharp
using Microsoft.EntityFrameworkCore;

public class Customer {
    public int Id { get; set; }
    public string Name { get; set; }
}

public class AppDbContext : DbContext {
    public DbSet<Customer> Customers { get; set; }

    protected override void OnConfiguring(DbContextOptionsBuilder optionsBuilder) {
        optionsBuilder.UseSqlServer("YourConnectionStringHere");
    }
}

// Adding a customer
using (var context = new AppDbContext()) {
    var customer = new Customer { Name = "John Doe" };
    context.Customers.Add(customer);
    context.SaveChanges();
}

// Querying customers
using (var context = new AppDbContext()) {
    var customers = context.Customers.ToList();
    foreach (var c in customers) {
        Console.WriteLine(c.Name);
    }
}
```

**Output:**

```text
John Doe
```

## 15.1 Writing Unit Tests with MSTest and NUnit

## 15.2 Mocking with Moq

Mocking бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸ objects бһ”бҹ’бһӣбҹӮбһҖбһҠбҹӮбһӣбһҮбҹҶбһ“бһҪбһҹ dependencies бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ unit testsбҹ” Moq бһӮбһәбһҮбһ¶ library бһҠбҹҸбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ mocking бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ C#бҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ mocking бһҮбһ¶бһҳбһҪбһҷ Moqбҹ–

```csharp
using Moq;

public interface ICalculator {
    int Add(int a, int b);
}

[TestClass]
public class CalculatorTests {
    [TestMethod]
    public void Add_TwoNumbers_ReturnsSum() {
        // Arrange
        var mockCalculator = new Mock<ICalculator>();
        mockCalculator.Setup(x => x.Add(5, 3)).Returns(8);

        // Act
        int result = mockCalculator.Object.Add(5, 3);

        // Assert
        Assert.AreEqual(8, result);
    }
}
```

## 15.3 Test-Driven Development (TDD) in C#

## 16.1 Reflection in C#

Reflection бһӮбһәбһҮбһ¶бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ metadata бһ“бҹғ types, methods, properties, бһ“бһ·бһ„ fields бһ“бҹ… runtimeбҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ reflectionбҹ–

```csharp
using System.Reflection;

Type type = typeof(string);
Console.WriteLine("Type Name: " + type.Name);

MethodInfo[] methods = type.GetMethods();
foreach (var method in methods) {
    Console.WriteLine("Method: " + method.Name);
}
```

## 16.3 Dependency Injection (DI)

Dependency Injection (DI) бһӮбһәбһҮбһ¶бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ dependencies бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ” бһңбһ¶бһҮбһҪбһҷбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһјбһҠбһҖбһ¶бһ“бҹӢбһҸбҹӮбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸбһ”бһ¶бһ“бһ“бһ·бһ„бһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“бҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ dependency injectionбҹ–

```csharp
public interface IMessageService {
    void SendMessage(string message);
}

public class EmailService : IMessageService {
    public void SendMessage(string message) {
        Console.WriteLine("Email sent: " + message);
    }
}

public class Notification {
    private readonly IMessageService _messageService;

    public Notification(IMessageService messageService) {
        _messageService = messageService;
    }

    public void Notify(string message) {
        _messageService.SendMessage(message);
    }
}

// Registering services in ASP.NET Core
public void ConfigureServices(IServiceCollection services) {
    services.AddTransient<IMessageService, EmailService>();
}
```

## 16.4 Design Patterns (Singleton, Factory, Observer)

Design patterns бһӮбһәбһҮбһ¶бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһ‘бһјбһ‘бҹ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ” бһ–бһҪбһҖбһңбһ¶бһҮбһҪбһҷбһ’бҹ’бһңбһҫбһұбҹ’бһҷбһҖбһјбһҠбһҖбһ¶бһ“бҹӢбһҸбҹӮбһўбһ¶бһ…бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ”бһ¶бһ“бһ“бһ·бһ„бһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“бҹ”

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹғ design patternsбҹ–

```csharp
// Singleton Pattern
public class Singleton {
    private static Singleton _instance;
    private Singleton() {}

    public static Singleton Instance {
        get {
            if (_instance == null) {
                _instance = new Singleton();
            }
            return _instance;
        }
    }
}

// Factory Pattern
public interface IProduct {
    void Create();
}

public class ProductA : IProduct {
    public void Create() {
        Console.WriteLine("Product A created.");
    }
}

public class ProductB : IProduct {
    public void Create() {
        Console.WriteLine("Product B created.");
    }
}

public class ProductFactory {
    public IProduct CreateProduct(string type) {
        switch (type) {
            case "A": return new ProductA();
            case "B": return new ProductB();
            default: throw new ArgumentException("Invalid product type.");
        }
    }
}

// Observer Pattern
public class Subject {
    private List<IObserver> _observers = new List<IObserver>();

    public void Attach(IObserver observer) {
        _observers.Add(observer);
    }

    public void Notify() {
        foreach (var observer in _observers) {
            observer.Update();
        }
    }
}

public interface IObserver {
    void Update();
}

public class Observer : IObserver {
    public void Update() {
        Console.WriteLine("Observer updated.");
    }
}
```
